Prerequisites:
1.	Windows Power Shell
2.	Python 3.0 or above versions
3.	Install requests module using pip install requests command in Power Shell.

Instructions to run the program:
1.	Open power shell from start menu.
2.	Navigate to the folder containing the python file using cd command.
	e.g. If the file is located in downloads folder then the command looks like below
	C:\Users\<username>\Downloads
3.	After navigating to the folder use below command to run the file.
 	python Task1.py
4.	After successful execution of script, enter latitude and longitudes in terminal when prompted.
5.	The user now can see the Atmospheric pressure for past 3 days.
